## 🚀 快速部署（外网可访问）

### 方案一：Vercel（推荐，免费，最快）

1. 访问 [vercel.com](https://vercel.com) 注册/登录（可以用 GitHub 账号）
2. 点击 **"Add New"** → **"Project"**
3. 选择 **"Continue without Git"**（或者上传到 GitHub 再导入）
4. 选择项目文件夹 `grade4-preview/`
5. 点击 **"Deploy"**，等待一分钟
6. 部署完成！会得到一个类似 `grade4-preview.vercel.app` 的网址

📱 手机和电脑都能访问，分享给孩子就行！

### 方案二：Netlify（也非常简单）

1. 访问 [netlify.com](https://netlify.com) 注册/登录
2. 把 `grade4-preview/` 文件夹拖到浏览器窗口
3. 等待几秒钟，自动部署完成
4. 得到一个 `xxx.netlify.app` 的网址

### 方案三：GitHub Pages

1. 在 GitHub 上创建一个仓库
2. 把项目文件上传
3. 进入 Settings → Pages → 选择 main 分支
4. 等待几分钟，访问 `你的用户名.github.io/仓库名`
